# Plan 07-01: Remove User `ta`

## Objective
Completely remove user `ta` from the nix-config repository.

## Context
The `ta` user was created for testing/secondary purposes but is no longer needed. All traces should be removed to clean up the config.

## Files to Remove/Modify

```
home/ta/                          # Entire directory (8 files)
├── common/nixos.nix
├── common/default.nix
├── genoa.nix
├── gusto.nix
├── grief.nix
├── iso.nix

hosts/common/users/ta/            # Entire directory
```

### Files to Modify
Search for and remove `ta` references in:
- `flake.nix` - home-manager user configurations
- `hosts/nixos/*/default.nix` - any hosts importing ta user
- Any other files referencing `ta`

## Steps

1. **Grep for all `ta` references**
   ```bash
   grep -r "\"ta\"" --include="*.nix" .
   grep -r "/ta" --include="*.nix" .
   grep -r "ta@" --include="*.nix" .
   ```

2. **Remove home/ta directory**
   ```bash
   rm -rf home/ta
   ```

3. **Remove hosts/common/users/ta directory**
   ```bash
   rm -rf hosts/common/users/ta
   ```

4. **Update flake.nix**
   - Remove ta from homeConfigurations
   - Remove ta from any user lists

5. **Update host configs**
   - Remove ta user imports from each host

6. **Verify build**
   ```bash
   nix flake check
   nh os build
   ```
